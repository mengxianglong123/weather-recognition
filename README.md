# 🔥🔥🔥Pytorch实战天气图片识别
## 代码目录结构
```shell
log --训练日志
model --模型存储目录
config.py --配置文件
data_loader.py --自定义数据加载
model.py --模型构建文件
pridect.py -- 预测模块
train.py -- 训练模块 
```
